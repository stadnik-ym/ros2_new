## Компілятор/Cmake/Кодовий редактор не бачить хеддери, що робити?

Перевірте чи точно ви знаходитесь в середовищі Nix (в 98% випадках це саме через це).

```
nix develop

zeditor . // code . та будь який інший редактор коду  
```

Також може бути таке, що n-нний пакет просто не встановлений. Якщо це так - внесіть його в flake.nix (на момент 29.06.2026 для поточної версії проекту ВСІ необхідні пакети занесені в flake.nix)
