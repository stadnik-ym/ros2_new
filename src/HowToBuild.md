## Як забілдити проект?

В корні проекта запустити:

```
source artemka.sh
```

Потім запускаєте лаунч або окремий пакет. 

Якщо Ви працюєте в декількох терміналах - в усіх інших (тобто всі крім того де було запущено скрипт для збірки) 

```
source install/setup.bash
```
