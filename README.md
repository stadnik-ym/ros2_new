# `ros2_new` 

Репозиторій містить воркспейс ROS 2 для керування роботом, розгорнутий з використанням декларативного пакетного менеджера Nix. Використання Nix гарантує 100% відтворюваність оточення (компілятори, бібліотеки OpenCV, залежності ROS 2) на будь-якій Linux-системі без необхідності ручного встановлення пакетів.

---

## Стек технологій
* **Робототехніка:** ROS 2 Jazzy
* **Оточення та збірка:** Nix Flakes, Colcon, CMake
* **Мова розробки:** C++17/20
* **Комп'ютерний зір:** OpenCV 4.x (модуль ArUco)

---

## Структура проекту

```text
├── .docker/            # Докеризація через Nix 
├── .github/            # Пайплайни (GH Actions)
├── colcon_ws/          # Необхідно для нормальної роботи clangd 
├── src/                # Вихідний код ROS 2 пакетів
│   └── aruco_detector/ # Нода детекції ArUco маркерів (C++)
├── flake.nix           # Декларативний опис оточення та залежностей Nix
├── flake.lock          # Фіксовані версії пакетів (пін залежностей)
├── Dockerfile          # Поточна конфігурація контейнеризації
├── compose.yaml        # Оркестрація контейнерів
└── build.sh            # Скрипт автоматизації збірки та очищення воркспейсу
